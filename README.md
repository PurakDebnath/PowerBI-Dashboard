# PowerBI-Dashboard
This project presents an interactive Power BI dashboard created using survey data collected from data professionals across different roles and industries. The goal of the project is to analyze key aspects of the data profession such as job roles, salaries, programming language preferences, work-life balance, and career satisfaction.

# Project Overview
This project analyzes survey data collected from data professionals across different countries.
The dashboard provides a clear view of job roles, salaries, programming language preferences, work-life balance, salary satisfaction, and learning difficulty.
The main goal is to understand trends and challenges faced by data professionals using interactive Power BI visualizations.

# Business Objective
1. Understand salary trends across different data roles.
2. Analyze work-life balance and salary satisfaction.
3. Identify popular programming languages among data professionals.
4. Measure how difficult people find entering or working in the data field
5. Help students, freshers, and professionals make data-driven career decisions

# Tools and Technologies Used

- Power BI – Data visualization and dashboard creation
- Microsoft Excel / CSV – Data source and basic data cleaning
- Power Query – Data transformation and preparation
- DAX (Data Analysis Expressions) – Calculated measures and KPIs

# Key Insights
- Average age of survey takers is around 29.87 years
- Data Scientists earn the highest average salary among listed roles
- Python is the most preferred programming language
- Majority of respondents are from the United States and India
- Work-life balance rating is moderate (around 5.74 / 10)
- Salary happiness is relatively low (around 4.27 / 10)
- Most respondents find the data field Easy to Moderately Difficult

# Project Process
- Data Collection – Survey data collected in CSV format
- Data Cleaning – Removed null values and standardized columns using Power Query
- Data Transformation – Created calculated columns and measures
- Dashboard Design – Built interactive visuals like:
- Bar charts
- Tree maps
- Donut charts
- Gauge visuals
- Insight Generation – Interpreted results to answer business questions

# Value Delivered
- Helps students and freshers understand the real data job market
- Assists career planning by comparing roles and salaries
- Provides insights into employee satisfaction
- Demonstrates real-world Power BI skills for portfolio and interviews
- Acts as a decision-support tool for aspiring data professionals
